# Database-programming
MyComesticShop là dự án cửa hàng mỹ phẩm mẫu xây dựng trên .NET, cung cấp backend RESTful API để quản lý sản phẩm, danh mục, người dùng, giỏ hàng và đơn hàng. Dự án sử dụng ASP.NET Core và Entity Framework Core, dễ mở rộng, triển khai local hoặc Docker và tích hợp với frontend.
